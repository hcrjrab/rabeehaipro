# Rabeeh AI Agent Pro

## Purpose

Rabeeh AI Agent Pro is a local-first AI Digital Employee.

Everything should run locally using Ollama.

## Tech Stack

Backend

* Python 3.14
* FastAPI
* LangGraph
* LangChain
* Ollama

Frontend

* Next.js
* React
* TailwindCSS
* TypeScript

AI Models

* qwen2.5-coder:7b (coding)
* deepseek-r1:7b (reasoning)
* gemma3:4b (general assistant)

Database

* PostgreSQL
* ChromaDB

Goals

The assistant should eventually support:

* AI Chat
* Multi-Agent System
* Memory
* Browser Automation
* Computer Vision
* Screen Understanding
* Mouse & Keyboard Control
* VS Code Automation
* Linux Automation
* Business Assistant
* Voice Assistant

Coding Rules

* Never rewrite working code.
* Build one module at a time.
* Explain every change.
* Use clean architecture.
* Keep modules independent.
* Ask before deleting or restructuring files.
